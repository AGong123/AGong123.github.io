Learn To Code - Unit 2

In Learn To Code 2, we learned some new principles and expanded on the ones we learned about in Learn To Code 1. I have provided short descriptions about these new and expanded principles below, along with 2 snapshots of my code, and a video of it in action.

  • Variables, we now have more, different values we need to store to later change our code.
  • Types define the data these new variables can hold.
  • Initialization, we now have many new and different values to start a variable.
  • Parameters, we also have more of these to customize functions different behaviors.
  • World Building is how we would add different blocks, characters, and items into the map.
  • Arrays help organize multiple pieces of data in an ordered way.
