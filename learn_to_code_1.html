Learn To Code - Unit 1

In Learn To Code 1, we learned about how to use 9 of the basic coding principles. Below I have put 2 snapshots of my code, and a video of the coding working. I have also written short definitions about each of these 9 principles.

  • Commands tell the computer what to do.
  • For loops repeat actions a set number of times.
  • Conditionals run code based on true or false conditions.
  • Logical operators combine conditions to make decisions.
  • Variables store values that change in your code.
  • Types define the kind of data a variable holds.
  • Initialization sets the starting value for a variable.
  • Functions group code to perform a specific task.
  • Parameters are inputs passed to functions to customize their behavior.
